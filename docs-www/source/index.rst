ZQuest Classic documentation
============================

This is the home for ZQuest Classic's documentation.

ZQuest Classic is a free, open source engine for making games similar to the original NES Zelda.

* Main website: `https://zquestclassic.com <https://zquestclassic.com>`_
* GitHub: `https://github.com/ZQuestClassic/ZQuestClassic/ <https://github.com/ZQuestClassic/ZQuestClassic/>`_
* `Latest downloads <https://zquestclassic.com/releases/>`_
* `Changelog <https://zquestclassic.com/changelog/>`_
* `Community <https://zquestclassic.com/community/>`_
